تحميل البرنامج

يمكنك تحميل آخر إصدار من مختبر القرآن الكريم مجانًا.

البرنامج متاح حاليا لنظام Windows فقط، ونتطلع بإذن الله إلى إطلاقه لمزيد من أنظمة التشغيل.

تحميل البرنامج أو كود المصدر من GitHub

يمكنك تحميل أحدث إصدار تم إطلاقه من البرنامج أو كود المصدر بالانتقال مباشرةً إلى أحدث إصدار على GitHub إذا كان لديك خبرة في github.

الانتقال إلى GitHub

التحميل من رابط مباشر

إذا لم يكن لديك خبرة في github وتريد تحميل البرنامج برابط مباشر، فيمكنك النقر على "تحميل الآن" أدناه.

تحميل الآن

بعد التحميل، قم بفك ضغط الملف "QuranTester_Portable.zip"، وافتح المجلد "QuranTester"، وشغل الملف التنفيذي "QuranTester.exe" لبدء استخدام البرنامج.

إذا واجهت أي مشاكل في البرنامج، يرجى الاتصال بنا.
